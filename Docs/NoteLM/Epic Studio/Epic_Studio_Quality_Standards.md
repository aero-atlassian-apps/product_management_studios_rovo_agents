# Epic Studio — Standards de qualité

## Bonne Epic (gate minimal)
- Titre clair, orienté résultat.
- Résumé concis (pourquoi/quoi/résultat).
- Description structurée: contexte, objectifs, portée, hypothèses, risques, critères de succès.
- Liens stratégiques (objectifs/initiatives; OKR si disponibles — facultatif).
- Ready basique: acteurs, dépendances majeures, premiers résultats, métriques clés.

## Sorties (standards)
- Markdown prêt Jira; sections et titres cohérents.
- Bullets courtes, verbes d’action; ≤ 180 mots par bloc.
- Liens vérifiés ou signalés manquants.

## Publication
- Consentement explicite et unique; preuve après exécution (lien/clé).
- Conversion Markdown → ADF si requis par le connecteur.
